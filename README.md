# Redis

### 通用命令
- keys [pattern] , 遍历所有的key
- dbsize，计算key的总数
- exists key，是否存在某个key
- del key [key...]，删除key
- expire key seconds，设置key在seconds秒后过期
- ttl key，查看key的剩余过期时间
- persist key，去掉key的过期时间
- type key，判断key的数据类型
